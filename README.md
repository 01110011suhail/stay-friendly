# Hotel Booking System

A full-stack **Hotel Booking System** developed using **Java Spring Boot**, **React.js**, and **MySQL**. This application allows users to browse available rooms, make bookings, and manage reservations through a modern and user-friendly interface. The system also provides administrative features for managing hotels, rooms, and customer bookings.

## Features

* User registration and login authentication
* Browse available hotel rooms
* Real-time room booking and reservation management
* View booking history
* Admin panel for managing rooms, users, and bookings
* Responsive frontend built with React
* RESTful APIs powered by Spring Boot
* MySQL database for secure data storage

## Tech Stack

**Frontend**

* React.js
* HTML5
* CSS3
* JavaScript

**Backend**

* Java
* Spring Boot
* Spring Data JPA
* REST APIs

**Database**

* MySQL

## Architecture

The project follows a **three-tier architecture**:

* **Frontend (React)** – Handles user interface and user interactions
* **Backend (Spring Boot)** – Business logic and API services
* **Database (MySQL)** – Stores user, booking, and room data

## Use Cases

* Customers can search and book hotel rooms online
* Admin can manage hotel listings and bookings
* Users can view and manage their reservations
